## 🎯 **КАК ЭТО РАБОТАЕТ В ПРОЕКТЕ:**

### **1. При оформлении заказа:**

```javascript
// Пользователь вводит город

// Сервер автоматически:
1. Вызывает calculateAllDeliveryOptions(city, weight)
2. СДЭК → делает реальный API запрос → получает тарифы
3. Boxberry → рассчитывает по формуле → возвращает цену
4. Деловые Линии → рассчитывает → возвращает цену
5. Яндекс → проверяет доступность → возвращает варианты

  delline: { terminal: { price: 450, days: 5 }, courier: { price: 680, days: 4 } },
  yandex: { express: { price: 520, days: 1 }, standard: { price: 220, days: 2 } }
}
```
