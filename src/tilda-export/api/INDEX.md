## 📐 Расчеты сеток

### Формула для колонок:
```
column_width = (content_area - gap × (n-1)) / n

где:
  content_area = 1120px (1200 - 80px padding)
  n = количество колонок
  gap = расстояние между колонками
```

### Примеры:
```
3 колонки, gap 48px:
  (1120 - 48×2) / 3 = 341.33px ≈ 340px

3 колонки, gap 32px:
  (1120 - 32×2) / 3 = 352px

4 колонки, gap 40px:
  (1120 - 40×3) / 4 = 260px
```

## 🔧 Как использовать файлы

### Вариант A: Ручная загрузка (рекомендуется для начинающих)

1. Откройте Tilda → Создайте страницу
2. Добавьте 4 блока Zero Block
3. Скопируйте HTML из каждого файла в соответствующий блок
4. Замените placeholder изображения
5. Опубликуйте

**Время:** 15-30 минут

### Вариант B: Через API (для опытных)

1. Получите API ключи в Tilda
2. Используйте cURL или JavaScript для загрузки
3. Примеры команд в `README_API.md`

**Время:** 5-10 минут
