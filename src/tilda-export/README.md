# 🎨 АЗЬ ЕСЬМ - Экспорт для Tilda Publishing

Готовый пакет для переноса стиля и компонентов вашего бренда в Tilda с использованием **Zero Block**.

---

## 📦 Что входит в пакет

### 1. **brand-variables.css** 
CSS файл с полной палитрой бренда, типографикой и готовыми классами

### 2. **hero-section.html**
Главная героическая секция с логотипом и призывом к действию

### 3. **philosophy-section.html**
Секция философии бренда с тремя колонками (Корни, Душа, Единство)

### 4. **product-card.html**
Сетка карточек товаров с hover эффектами

---

## 🚀 Быстрый старт

### ШАГ 1: Подключите CSS переменные

1. Откройте ваш сайт в Tilda
2. Перейдите в **Настройки сайта** → **Еще** → **HTML код для вставки внутри HEAD**
3. Скопируйте содержимое файла `brand-variables.css`
4. Вставьте в тег `<style>...</style>`:

```html
<style>
  /* Вставьте сюда содержимое brand-variables.css */
</style>
```

5. Сохраните настройки

---

### ШАГ 2: Создайте страницу с Zero Block

1. В Tilda создайте **новую страницу**
2. Нажмите **+ Добавить блок**
3. Выберите **Другое** → **T123 - Zero Block**
4. Нажмите **Настройки блока** (шестеренка)

---

### ШАГ 3: Добавьте компоненты

#### Для Hero секции:

1. В Zero Block нажмите **HTML**
2. Скопируйте всё содержимое файла `hero-section.html`
3. Вставьте в HTML редактор
4. **Важно:** Замените плейсхолдеры:
   - `URL_ВАШЕГО_ЛОГОТИПА` → загрузите логотип в Tilda и вставьте URL
   - `URL_ФОНОВОГО_ИЗОБРАЖЕНИЯ` → загрузите фоновое фото и вставьте URL
5. Сохраните

#### Для секции философии:

1. Создайте **новый Zero Block**
2. Скопируйте содержимое `philosophy-section.html`
3. Вставьте в HTML редактор
4. Сохраните

#### Для товаров:

1. Создайте **новый Zero Block**
2. Скопируйте содержимое `product-card.html`
3. Вставьте в HTML редактор
4. Замените:
   - URL изображений товаров
   - Названия, описания, цены
5. Сохраните

## 📐 Типографика

### Шрифты:
- **Заголовки:** Georgia, Times New Roman (serif)
- **Текст:** системный шрифт (по умолчанию Tilda)

### Размеры:
- **Hero заголовок:** 96px (desktop) / 56px (mobile)
- **H1:** 64px
- **H2:** 48px
- **H3:** 32px
- **Основной текст:** 16px

### Межбуквенное расстояние:
- **Заголовки:** `letter-spacing: 0.1em - 0.2em`
- **Основной текст:** `letter-spacing: 0`

## 🖼️ Требования к изображениям

### Логотип:
- Формат: PNG с прозрачностью
- Размер: 500x500px (минимум)
- Вес: до 100 КБ

### Фоновое изображение Hero:
- Формат: JPG
- Размер: 1920x1080px (минимум)
- Вес: до 500 КБ
- Композиция: центральная или с акцентом

### Изображения товаров:
- Формат: JPG
- Соотношение: 3:4 (например, 600x800px)
- Вес: до 200 КБ на фото
- Фон: однотонный (светлый или темный)

## 🔧 Кастомизация

### Изменить цвета:

В файле `brand-variables.css` найдите раздел `:root` и измените значения:

```css
:root {
  --brand-gold: #D4AF37;  /* Ваш новый золотой */
  --brand-red: #C41E3A;   /* Ваш новый красный */
  /* и т.д. */
}
```

### Изменить размеры текста:

```css
:root {
  --text-hero: 5rem;  /* Было 6rem */
  --text-h1: 3.5rem;  /* Было 4rem */
}
```

### Добавить больше товаров:

В файле `product-card.html` скопируйте блок:

```html
<article class="product-card">
  <!-- ... -->
</article>
```

И вставьте внутри `<div class="products-azezm__grid">`, заменив содержимое.

Все компоненты **полностью адаптивны** и работают на:

## ⚡ Оптимизация

### Скорость загрузки:

1. **Сжимайте изображения** перед загрузкой:
   - Используйте TinyPNG.com или Squoosh.app
   - Целевой размер: 100-200 КБ на изображение

2. **Минифицируйте CSS** (опционально):
   - Используйте cssminifier.com
   - Вставьте минифицированную версию в Tilda

3. **Lazy loading для изображений**:
   - Tilda автоматически добавляет это для Zero Block

## 🆘 Частые проблемы

### ❌ Логотип не отображается
**Решение:** Проверьте, что URL логотипа:
1. Начинается с `https://`
2. Заканчивается на `.png` или `.jpg`
3. Доступен (откройте URL в новой вкладке)

### ❌ Стили не применяются
**Решение:** 
1. Убедитесь, что `brand-variables.css` вставлен в HEAD сайта
2. Очистите кеш браузера (Ctrl+Shift+R)
3. Проверьте, что в Zero Block HTML вставлен полностью

### ❌ Секция не адаптивна на мобильных
**Решение:**
1. В Zero Block зайдите в **Настройки** → **Адаптив**
2. Включите опцию "Адаптировать для мобильных"

### ❌ Изображения загружаются медленно
**Решение:**
1. Сжимайте изображения до 100-200 КБ
2. Используйте формат WebP (конвертируйте на squoosh.app)
3. В Tilda включите "Оптимизацию изображений"

## 🎓 Советы по работе с Zero Block

### 1. **Всегда используйте контейнеры**
Оборачивайте контент в `.brand-container` для правильных отступов:

```html
<section class="brand-bg-navy brand-section">
  <div class="brand-container">
    <!-- Ваш контент -->
  </div>
</section>
```

### 2. **Тестируйте на разных устройствах**
В Tilda есть кнопка "Предпросмотр" → выберите разные устройства

### 3. **Сохраняйте копии**
Перед большими изменениями делайте копию страницы (Настройки → Дублировать)

### 4. **Используйте комментарии в коде**
Помечайте секции комментариями для лёгкой навигации:

```html
<!-- НАЧАЛО: Секция товаров -->
<section class="products-azezm">
  ...
</section>
<!-- КОНЕЦ: Секция товаров -->
```
