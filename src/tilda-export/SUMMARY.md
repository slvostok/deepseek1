## 🎯 Ключевые параметры

### Сетки (точные расчеты)

**Frame 2: Philosophy (3 колонки)**
```
Ширина колонки: 340px
Gap: 48px
Формула: (1120 - 48×2) / 3 = 341px ≈ 340px
```

**Frame 3: Products (3 колонки)**
```
Ширина карточки: 352px
Высота изображения: 469px (aspect 3:4)
Gap: 32px
Формула: (1120 - 32×2) / 3 = 352px
```

**Frame 4: Footer (4 колонки)**
```
Ширина колонки: 260px
Gap: 40px
Формула: (1120 - 40×3) / 4 = 260px
```

## 🚀 Как использовать

### Вариант А: Ручная загрузка (рекомендуется)
```
1. Tilda → Создать страницу
2. Добавить 4 блока Zero Block
3. Скопировать HTML из каждого frame-0X-*.html
4. Заменить placeholder изображения
5. Опубликовать

Время: 15-30 минут
```

### Вариант Б: Через API
```
1. Получить API ключи в Tilda
2. Использовать cURL или JS для загрузки
3. Примеры в /api/README_API.md

Время: 5-10 минут
```
