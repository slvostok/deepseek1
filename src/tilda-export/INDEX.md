# 📦 АЗЬ ЕСЬМ - Tilda Export Package

Полный пакет для переноса стиля бренда в Tilda Publishing через Zero Block.

```
tilda-export/
```

## 🔥 НОВИНКА: Готовые фреймы для Tilda API

### ✅ Все требования выполнены:
1. **Ширина фреймов: 1200px** (точно!)
2. **Разбито на логичные фреймы** (4 секции)
3. **Ширина шейпов = ширина текста** (идеальное выравнивание)

### 📦 Что внутри `/api/`:
- ✅ 4 готовых HTML фрейма (1200px ширина)
- ✅ Точные расчеты сеток и колонок
- ✅ Полная конфигурация в JSON
- ✅ Инструкции по загрузке через API и вручную

**➡️ Начните с файла: `/api/README_API.md`**

## 📋 Чек-лист подготовки

Перед началом убедитесь, что у вас есть:

### Изображения:
- [ ] Логотип (PNG, прозрачный, ~500x500px)
- [ ] Фоновое фото для Hero (JPG, 1920x1080px)
- [ ] Фото товаров (JPG, 600x800px, минимум 3 шт)

### Контент:
- [ ] Тексты описаний товаров
- [ ] Цены товаров
- [ ] Email и соцсети для футера

### Доступы:
- [ ] Доступ к админке Tilda
- [ ] Возможность создавать Zero Block (не на бесплатном тарифе)

## 🔥 Популярные вопросы

### Можно ли использовать на бесплатном тарифе?
❌ Нет, Zero Block доступен с тарифа Personal

### Нужно ли знать HTML/CSS?
⚠️ Базовые знания помогут, но можно и без них (следуйте инструкции)

### Сколько времени займет?
⏱️ 10-30 минут в зависимости от опыта

### Будет ли работать на мобильных?
✅ Да, всё полностью адаптивно

### Можно ли менять цвета?
✅ Да, легко в файле `brand-variables.css`

### Можно ли добавить свои секции?
✅ Да, используйте готовые классы из CSS
