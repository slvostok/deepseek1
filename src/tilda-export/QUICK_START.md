# ⚡ БЫСТРЫЙ СТАРТ - АЗЬ ЕСЬМ в TILDA

Упрощенная инструкция для тех, кому нужно быстро всё настроить.

---

## 🎯 За 5 минут

### ШАГ 1: Подготовьте изображения (2 мин)

Вам нужно:
1. **Логотип** (PNG, прозрачный фон, ~500x500px)
2. **Фоновое фото** для Hero (JPG, 1920x1080px)
3. **3 фото товаров** (JPG, 600x800px каждое)

📥 **Оптимизируйте на:** https://tinypng.com/

---

### ШАГ 2: Загрузите в Tilda (1 мин)

1. В Tilda перейдите в **Файловый менеджер**
2. Загрузите все изображения
3. **Скопируйте URL** каждого изображения (ПКМ → Копировать адрес)

---

### ШАГ 3: Вставьте CSS (1 мин)

1. **Настройки сайта** → **Еще** → **HTML код для HEAD**
2. Откройте файл **`brand-variables.css`**
3. Скопируйте **ВСЁ** и оберните в теги:

```html
<style>
  /* Сюда вставьте весь код из brand-variables.css */
</style>
```

4. **Сохраните**

---

### ШАГ 4: Создайте страницу (1 мин)

1. **Создать новую страницу**
2. **+ Добавить блок** → **Другое** → **T123 Zero Block**
3. В Zero Block нажмите **HTML** (иконка `</>`)

---

### ШАГ 5: Вставьте компоненты (2 мин)

#### Hero секция:
1. Откройте **`hero-section.html`**
2. Скопируйте **всё**
3. Вставьте в HTML редактор Zero Block
4. **Найдите и замените:**
   - `URL_ВАШЕГО_ЛОГОТИПА` → вставьте URL логотипа из Шага 2
   - `URL_ФОНОВОГО_ИЗОБРАЖЕНИЯ` → вставьте URL фонового фото
5. **Сохраните**

#### Философия (опционально):
1. Создайте **новый Zero Block** ниже
2. Откройте **`philosophy-section.html`**
3. Скопируйте и вставьте
4. **Сохраните**

#### Товары (опционально):
1. Создайте **новый Zero Block** ниже
2. Откройте **`product-card.html`**
3. Скопируйте и вставьте
4. **Замените:**
   - URL изображений товаров (3 штуки)
   - Названия товаров
   - Цены
5. **Сохраните**

---

### ✅ ГОТОВО!

Нажмите **"Предпросмотр"** чтобы увидеть результат.

## 🔥 Только самое важное

### Что ОБЯЗАТЕЛЬНО нужно заменить:

В **каждом** HTML файле найдите эти строки и замените:

```html
<!-- В hero-section.html -->
src="URL_ВАШЕГО_ЛОГОТИПА"              → ваш логотип
style="background-image: url('URL_...')" → ваше фото

<!-- В product-card.html -->
src="https://images.unsplash.com/..."   → ваши фото товаров (3 шт)

<!-- В footer-section.html -->
src="URL_ВАШЕГО_ЛОГОТИПА"              → ваш логотип
href="mailto:info@azezm.ru"            → ваш email
href="https://instagram.com/azezm"     → ваш Instagram
```

### Как искать и заменять:
1. В редакторе HTML нажмите **Ctrl+F** (Windows) или **Cmd+F** (Mac)
2. Введите `URL_`
3. Замените на ваш реальный URL

## 🆘 Если что-то не работает

### Логотип не показывается:
- Проверьте, что URL начинается с `https://`
- Откройте URL в браузере - должно показать картинку

### Стили не применяются:
- Проверьте, что CSS вставлен в HEAD (Шаг 3)
- Очистите кеш: **Ctrl+Shift+R** (Windows) / **Cmd+Shift+R** (Mac)

### Секция выглядит криво:
- Убедитесь, что скопировали **весь** код из HTML файла
- Проверьте, что не удалили теги `<style>` или `</section>`
