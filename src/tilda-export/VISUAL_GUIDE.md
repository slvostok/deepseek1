## 📏 Формулы расчета

### Ширина колонок
```
formula = (content_width - gap × (columns - 1)) / columns

Примеры:
  3 колонки, gap 48: (1120 - 48×2) / 3 = 341.33 ≈ 340px
  3 колонки, gap 32: (1120 - 32×2) / 3 = 352px
  4 колонки, gap 40: (1120 - 40×3) / 4 = 260px
```

### Aspect Ratio 3:4
```
height = width × 4 / 3

Пример:
  Ширина 352px → Высота = 352 × 4 / 3 = 469.33 ≈ 469px
```

### Content Area
```
content_width = container_width - (padding_left + padding_right)

Пример:
  1200px - (40px + 40px) = 1120px
```
